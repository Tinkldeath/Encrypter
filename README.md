# Encrypter

## Requrements:
* iOS 16.0+ Support
* CRUD for files
* Random key generation

## Features: 
* AES and ChaChaPoly encryption algorythms
* Storing decryption keys in a Keychain
* iOS Files Support
* Clean Swift Architecture
* MVVM for presentation
* RxSwift and RxCocoa data binding

## Stack:
* UIKit
* Realm Swift SDK
* RxSwift
* RxCocoa
* CryptoKit

## Building:
* Clone repository
* Install pods
* Open .xcworkspace file
